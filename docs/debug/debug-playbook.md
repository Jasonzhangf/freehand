# Debug Playbook

## Start

1. open `docs/architecture/feature-map.md`
2. locate `feature_id`
3. confirm owner, `debug_artifacts`, `runtime_paths`
4. inspect runtime evidence under `~/.freehand`
5. capture semantic position and scene position together
6. replay before patch when possible
7. fix owner truth
8. if truth changed, update map/docs/skill/memory in same task

## Do Not

- debug from grep alone
- add ad hoc temp helpers in orchestrators
- invent random runtime directories
